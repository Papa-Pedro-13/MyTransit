# My Transit

Верстка сайта компании перевозчика с использованием анимаций при прокрутке с помощью плагина gsap.

<h2><b>Стек frontend технологий:</b> Javascript, HTML, SCSS.</h2>

<h3>Инструкция по запуску:</h3>

```
npm install
npm run dev
```
